# HealthAndDamageArea

A node that can be added to an enemy or npc that will manage damage and health.

## Installation

Make sure to download the ChillCube Developer tools: https://github.com/ChillCube/ChillCube-Developer-Tools.git

### Using the Godot Plugin
Enable the **ChillCube Tools** plugin, then paste the repo URL into the Clone tab.

### Linux / CLI
```bash
clone-gd-addon https://github.com/ChillCube/HealthAndDamageArea.git
```

## Documentation
https://github.com/ChillCube/HealthAndDamageArea/blob/main/DOCUMENTATION.md

## Deinstallation
Use the CC Tools panel in Godot, or run:
```bash
remove-gd-addon HealthAndDamageArea
```

## For Developers
```bash
push-all-addons
```

- [DamageFormula](https://github.com/ChillCube/DamageFormula)
